# BVCU-SDK-Demo

MCP和MPU所有的功能均基于BVCU SDK进行开发。   
内部封装了音视频编解码、网络发送和接收、音视频渲染、回声消除、降噪等功能。    
开发者可以用简单的十几行SDK调用开发出一个简单的类似MPU音视频发送和对讲功能以及类似MCU音视频接收预览功能。 

SDK主要功能接口如下：
- 音视频传输和对讲 
- MCU设备音视频预览 
- 远程文件检索、下载、上传（包括平台和设备） 
- 远程设备查询、配置 
- 群组创建、删除、配置、 
- 群组的文字、文件、GPS和表情等IM消息收发
